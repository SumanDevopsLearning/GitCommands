GIT Best Practices:
1. don't commit your code in middle of your work,
   commit your code once evetything is ready.
2. Test your code before commit.
3. Please give proper commit message.
4. Avoid Merge conflicts.
5. Raise a pull request yo merge the changes to main branch.
6. Always monitor the branches, delete the unused branches after approvals.
7. Delete stashed if not required.
